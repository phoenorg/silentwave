# SilentWave

SilentWave is an Android app that helps reduce tinnitus with carefully selected white noise and ambient sounds. Create a personalized sound environment for better sleep.


## Installation

Download the APK from [here](https://www.phoen.org/silentwave) section or build the app from source.
